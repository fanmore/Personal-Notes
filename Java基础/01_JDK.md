## JVM、JRE、JDK的区别

JVM
* Java Virtual Machine
* Java虚拟机，能识别.class文件，将class文件中的字节码指令进行识别并调用操作系统向上的API完成动作。

JRE
* Java Runtime Environment
* Java运行时环境，包含JVM的标准实现和Java基本类库。

JDK
* Java Development Kit
* Java开发工具包，集成了JRE和开发工具。

JVM、JRE、JDK有层层嵌套关系。
